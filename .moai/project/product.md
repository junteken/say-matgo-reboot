# 세이 맞고 리부트 - 제품 문서

## 프로젝트 개요

**프로젝트명:** 세이 맞고 리부트 (Say Mat-go Reboot)
**유형:** 웹 기반 2인용 실시간 카드 게임
**기술 스택:** Next.js + TypeScript + WebSocket + Vercel
**개발 모드:** TDD (Test-Driven Development)

세이 맞고 리부트는 전통적인 한국 맞고 게임을 현대적인 웹 환경으로 재탄생시킨 실시간 2인용 카드 게임입니다. AI 기반의 스마트 카드 추천, 다양한 특수 규칙, 그리고 화려한 아바타 시스템을 통해 사용자들에게 새로운 게임 경험을 제공합니다.

## 타겟 사용자

### 1차 타겟
- **전통 맞고 게임爱好者:** 한국 전통 게임의 재미를 선호하는 20-40대 사용자
- **실시간 대전 게임 플레이어:** 빠른 속도와 전략적인 플레이를 즐기는 사용자
- **소셜 게임 사용자:** 아바트 커스터마이징과 순위 경쟁을 즐기는 사용자

### 2차 타겟
- **카드 게임 입문자:** 간단한 규칙과 튜토리얼로 게임을 배우고 싶은 초보자
- **모바일 게임 전환자:** PC에서 모바일로 옮겨온 경쟁형 게임 사용자
- **e스포츠 관중:** 전통 게임의 현대적 재해석에 관심 있는 e스포츠 팬

## 핵심 기능

### 1. 게임 시스템
- **기본 규칙:** 7점 이상 달성 시 "Go" 선언 또는 "Stop" 선택
- **특수 메커니즘:**
  - 폭탄 (Bomb): 특정 조건에서 추가 점수 획득
  - 쪽/따닥/싹쓸이: 특정 패턴으로 상대방의 카드를 획득
  - 총통 (Chongtong): 게임의 승패를 뒤바꾸는 강력한 효과
- **배수 시스템:** Go 수에 따른 보너스 (1고, 2고, 3고 2배, 4고 4배...)
- **페널티 시스템:** 피박, 광박, 멍박, 고박 등 다양한 페널티

### 2. 아바타 시스템
- **풀바디 아바타:** 다양한 표정과 애니메이션을 갖춘 전신 아바타
- **감정 반응:**
  - 상대방 카드 훔치기 성공 시 만족스러운 표정
  - "뻑" 당했을 당황스러운 표정
  - "Go" 선언 시 자신만만한 표정
- **아바트 커스터마이징:** 다양의 스킨, 액세서리, 배경 선택 가능

### 3. 랜덤 미션 시스템
- **게임별 미션:** 각 게임 시작 시 중앙에 표시되는 특별 미션
  - 예: "홍단 2배", "첫 끗발 3배", "연속 Go 성공"
- **미션 보상:** 미션 달성 시 추가 보너스 포인트 또는 아이템 획득
- **미션 난이도:** 초급, 중급, 고급 단계별 난이도 조절

### 4. 순위 시스템
- **자산 기반 진화:**
  - 평민 → 거상 → 영주 → 국왕
- **랭킹 기준:** 총 자산, 승률, 최고 Go 기록, 연속 승리 수
- **시즌 랭킹:** 매월 초기화되는 시즌별 순위 시스템
- **특수 칭호:** 랭��� 달성 시 획득할 수 있는 다양한 칭호

### 5. 데이터베이스 시스템
- **users:** 사용자 기본 정보 (닉네임, 레벨, 경험치, 현재 아바트)
- **user_assets:** 사용자 자산 (돈, 루비, 일일 충전 횟수)
- **user_stats:** 사용자 통계 (총 게임 수, 승률, 최고 상금, 최고 Go 수)
- **match_history:** 매치 기록 (승자, 패자, 기본 금액, 최종 점수, 배수, 승리 유형)

## 사용 사례

### 1. 친구와의 빠른 대전
- 사용자가 로그인하면 즉시 친구와 매칭 가능
- 5분 내로 게임 시작 가능한 빠른 매칭 시스템
- 실시간 채팅 기능으로 대화하며 게임 가능

### 2. AI와의 연습 모드
- 실시간 대신 AI와 1:1 연습 가능
- 난이도 조절 (초급, 중급, 고급 AI)
- 게임 후 전략 분석 및 팁 제공

### 3. 토너먼트 참여
- 주간/월간 토너먼트 참여
- 참여비와 보상이 설정된 공정한 토너먼트
- 실시간 순위 갱신 및 시상식

### 4. 아바트 커스터마이징
- 게임으로 획득한 포인트로 아바트 구매 가능
- 한정판 스킨 이벤트 참여
- 아바트 조합으로 개성 표현

### 5. 통계 및 분석
- 게임 기록 상세 분석
- 개인 강/약점 분석
- 전략적 개선을 위한 데이터 제공
