## [1.0.0] - 2026-02-28

### 추가된 기능 (Added)

#### 🎮 게임 로직 코어 구현 (SPEC-GAME-001)

- **CardDeck 모듈**
  - 48장의 화투 카드 덱 생성 및 관리
  - Fisher-Yates 알고리즘을 사용한 셔플
  - 시드 기반 재현 가능한 게임 지원
  - 카드 분배 및 잔여 카드 확인 기능

- **CardMatcher 모듈**
  - 기본 매칭 로직 구현 (월별 매칭)
  - 쪽(쪼그) 자동 판정 시스템
  - 카드 플레이 결과 처리 (매칭, 추가, 획득)
  - 매칭 가능 여부 확인 메서드

- **CardScorer 모듈**
  - 완전한 점수 계산 시스템
  - 광, 열끗, 띠, 피 별 점수 계산
  - 특수 조합 확인 (비광, 홍단, 청단 등)
  - 고 배수 계산 기능

- **GoStopSystem 모듈**
  - 고 선언 조건 확인
  - 고 선언 및 스톱 처리
  - 배수 시스템 (1고~5고)
  - 게임 상태 관리

- **PenaltyRules 모듈**
  - 4가지 페널티 규칙 구현
    - 피박: 승자 피 10장+, 패자 피 0장 (-2점)
    - 광박: 승자 광 3장+, 패자 광 0장 (-3점)
    - 멍박: 승자 열끗 획득, 패자 열끗 0장 (-2점)
    - 고박: 고 선언 후 추가 득점 없이 스톱 (-2점)
  - 페널티 자동 판정 및 점수 계산
  - 페널티 적용 여부 확인

#### 🧪 테스트 시스템

- 82개의 단위 테스트 구현
- 모든 핵심 모듈 100% 테스트 커버리지
- Vitest 테스트 프레임워크 통합
- TDD 개발 방식 적용

#### 📚 문서화

- 게임 로직 코어 API 문서 생성 (`docs/api/game-core.md`)
- 상세한 사용 예제 및 설명

#### 🏗️ 아키텍처

- 독립적인 모듈 간 의존성 관리
